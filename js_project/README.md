# Система управления начислениями за коммунальные услуги

Система для управления начислениями за коммунальные услуги с генерацией извещений на оплату в формате Excel.

## Описание

Проект реализует систему учета коммунальных услуг, включающую:

- Управление данными о гражданах, лицевых счетах и адресах
- Учет услуг и тарифов
- Начисление платежей с поддержкой валидации и применения скидок
- Генерацию извещений на оплату в формате Excel

Система использует объектно-ориентированный подход с применением паттерна проектирования Chain of Responsibility для обработки начислений.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Или напрямую:

```bash
node main.js
```

Программа сгенерирует Excel-файлы с извещениями на оплату для всех лицевых счетов в базе данных.

## Структура проекта

- `models.js` - Модели данных (Street, PersonalAccount, Service, Charge, PaymentNotice)
- `database.js` - Класс для хранения и управления данными
- `dateTimeUtils.js` - Утилиты для работы с датой и временем
- `chainOfResponsibility.js` - Реализация паттерна Chain of Responsibility для обработки начислений
- `excelGenerator.js` - Генератор Excel-файлов
- `main.js` - Главный модуль программы

## Технологии

- JavaScript (ES6+)
- Node.js
- ExcelJS - библиотека для работы с Excel файлами
